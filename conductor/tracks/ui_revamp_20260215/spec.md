# Specification - Full UI/UX Revamp

## Overview
Transform the Zone Photo Tracker from a basic interface into a modern, high-performance industrial application. This revamp integrates Tailwind CSS and Shadcn/UI to provide a robust framework, styled with a sophisticated "Refined Glassmorphic Industrial" aesthetic.

## Functional Requirements
- **Framework Integration:** Install and configure Tailwind CSS and Shadcn/UI components.
- **Global Navigation & Layout:**
    - Unified header with role-based visibility.
    - Improved user settings and logout experience.
    - Implementation of a global Dark Mode toggle.
- **Operations Center (Dashboard):**
    - High-density grid layout for zones.
    - Interactive SVG/Canvas charts for project statistics (CEO/Manager roles).
    - Refined status badges with improved color theory for accessibility.
- **Inspection Records (Zone Detail):**
    - Drag-and-drop photo upload interface.
    - Streamlined, mobile-optimized photo comparison view.
    - Real-time visual progress indicators for the documentation cycle.
- **Field-Ready Design:**
    - Larger touch targets (minimum 44x44px) for mobile users.
    - High-contrast typography and borders for visibility in harsh lighting.

## Non-Functional Requirements
- **Responsive Design:** Seamless transition between desktop, tablet, and mobile devices.
- **Performance:** Maintain fast load times despite enhanced animations and depth effects.
- **Accessibility:** Ensure high contrast ratios and keyboard navigability.

## Acceptance Criteria
- Full transformation of the Login, Dashboard, and Zone Detail pages.
- Dark mode functional across all screens.
- Mobile experience verified for ease of use in industrial settings.
- Interactive charts correctly reflect live zone data.
