# Technology Stack - Zone Photo Tracker

## Frontend
- **Framework:** Next.js 14 (App Router)
- **Library:** React 18
- **UI Components:** Shadcn/UI & Lucide React (Icons)
- **Animations:** Framer Motion
- **Styling:** Tailwind CSS & CSS3 (Globals & Modules)

## Backend
- **Environment:** Node.js
- **Framework:** Next.js API Routes (Serverless)
- **Authentication:** NextAuth.js
- **File Parsing:** Multer

## Database
- **Platform:** MongoDB Atlas
- **Driver:** `mongodb` native driver

## Utilities
- **Date/Time:** Moment.js
- **ID Generation:** UUID (v4)
- **Automated Communication:** Standardized notification and user utility modules leveraging Nodemailer.
- **Data Visualization:** Recharts for interactive SVG charting and analytics.
- **Reporting:** jsPDF and html2canvas
- **Email:** Nodemailer
